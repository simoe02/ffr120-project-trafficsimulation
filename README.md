# FFR120 Project - Traffic Simulation of Gothenburg

Source code for a project in the course FFR120 Simulation of complex systems. Includes all necessary data and code to run the simulation and produce the same figures as in the report.

If a venv with the listed packages in requirements.txt is used it should just be to run main.py to run the simulation with visualization.

For actual data gathering I used PyPy3 to run it much faster. The program is not very optimized. PyPy however does not support pyqtgraph so it cannot run with visualization.
